# 核心
不是跟读，不是听一句，暂停一下，复述一遍，而是刚听到第一次单词的时候就开始复述。
重点是同步反应，训练耳朵快速抓住内容，同时，嘴巴能够跟上节奏。
# 设备
蓝牙耳机
材料：
按水平分 3 个级别：
1. 基础 < 3000：选择慢速英语 VOA Special English / BBC Learning English 的 six minutes English / https://breakingnewsenglish.com/ 或者 Peppa PIG，模仿音调与停顿
2. 进阶 3000 <> 5000: 对话类，FRIENDS & Modern Family～ 短一些，1 到 2 分钟，或者演讲类，比如 TED 的 ED 版本
3. 高阶 > 5000 TED 演讲

# 环境
10 分钟即可

# 具体操作
1. 入门级： 盲跟与找节奏，不纠结对错，重点是跟上音频的节奏与语调。
	1. 先听一遍，大概知道内容
	2. 第二遍开始盲跟，没有跟上也不要停，重点是跟上节奏，让耳朵与嘴巴形成初步的配合
2. 基础级：对着文本来纠错，找出没有跟上的地方，光盲跟上不够的，需要文本材料
	1. 先盲跟一遍，录下自己的声音。
	2. 听自己的声音，在文本中标记没有跟对的地方，错误的地方反复对比音频中的发音，一直练习到正确。
	3. 再重复跟读，录音
	4. 再对比文本，纠错
3. 进阶级：用自己的语言来复述表达
# 坑
1. 跟不上，越跟越慌，想放弃 --> 先练简单句子
2. 发音总错，自己听不出来 --> 对比法，把自己的录音与音频放一起，逐句对比
3. 没时间 -> 融入生活
4. 跟的很熟，但自己说话还是不会用 -> 经常使用


- 不要追求完美，跟不上，跟不对非常正常，只追求坚持！！！
